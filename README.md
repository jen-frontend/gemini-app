## 소개
- Gemini API를 활용한 채팅 AI 서비스

## 사용 기술
- Frontend: React 19, Zustand, react-router-dom, Webpack, Babel, Firebase Authentication
- Backend: Express.js, Node.js
- DB: Firebase, Firestore
- AI: @google/genai 0.7.0
- Styling: Module CSS
- Deploy: Render
